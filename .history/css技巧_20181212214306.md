## 怎样写出更好的CSS代码呢？
### 1. line-height 使用相对大小
```
/*good:*/
font-size: 20px;
line-height: 30px;

/*bad*/
font-size: 20px;
line-height: 1.5;
```
### 2. 在适当的位置使用相对单位em，rem，vw，vh，vmin，vmax，（相对视口宽高的百分比）%
